# Calculator Demo

A simple command-line calculator tool written in Python that supports basic arithmetic operations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Error handling for invalid inputs and division by zero

## Requirements

- Python 3.6 or higher
- No external dependencies (uses only Python standard library)

## Usage

Run the calculator:

```bash
python main.py
```

Enter calculations in the format: `number operator number`

### Examples

```
>>> 5 + 3
Result: 8

>>> 10 - 4
Result: 6

>>> 7 * 6
Result: 42

>>> 15 / 3
Result: 5.0

>>> 10 / 0
Error: Cannot divide by zero
```

Type `q` to quit the calculator.
